# HireFlow

HireFlow is a role-based interview management platform designed to simulate a modern recruitment workflow.

## Features

- Admin, Interviewer and Candidate roles
- Candidate assignment system
- Interview scheduling
- Audit logging
- Secure authentication
- Luxury dark themed UI

## Tech Stack

- Node.js
- Express.js
- MongoDB
- EJS
- Tailwind CSS

## Workflow

1. Candidate registers into the platform  
2. Admin assigns interviewer  
3. Interview schedule is created  
4. Interviewer updates status  
5. Audit logs store every action
